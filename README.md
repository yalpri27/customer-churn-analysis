# Customer Churn Analysis & Prediction

## Project Overview

Customer Churn Analysis & Prediction is an end-to-end data analytics and machine learning project focused on understanding customer behavior, identifying churn patterns, and predicting customers who are likely to leave. The project covers the complete analytics workflow, including data cleaning, exploratory data analysis (EDA), customer segmentation, predictive modeling, business recommendations, and interactive dashboard development.

Using Python and Machine Learning techniques, customer data was analyzed to uncover the key factors influencing churn. A predictive model was built to identify high-risk customers, enabling proactive retention strategies. Additionally, customer segmentation was performed to better understand different customer groups and their behavior patterns.

To support business decision-making, an interactive Power BI dashboard was developed featuring KPIs, churn rate analysis, demographic insights, regional comparisons, product usage trends, and customer retention metrics. The project demonstrates how data-driven insights can help organizations reduce customer attrition, improve customer satisfaction, and increase long-term business value.

🎯 Project Objectives
Identify key factors driving customer churn.
Analyze customer behavior through EDA and segmentation.
Build a machine learning model to predict churn probability.
Generate actionable business recommendations.
Create an interactive Power BI dashboard for stakeholder insights.
