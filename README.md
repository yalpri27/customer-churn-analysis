# Customer Churn Analysis & Prediction

## Project Overview

This project analyzes customer behavior and predicts customer churn using Machine Learning and Power BI.

## Project Workflow

Raw Data
↓
Data Cleaning
↓
EDA Analysis
↓
Customer Segmentation
↓
Churn Prediction
↓
Business Recommendations
↓
Power BI Dashboard

## Tools Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- Power BI
- Matplotlib
- Seaborn

## Key Insights

- Customers with only one product have the highest churn.
- Inactive customers are more likely to leave.
- Germany shows relatively higher churn.
- Customer engagement strongly impacts retention.
